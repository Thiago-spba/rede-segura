#  Master Security  Análise de Redes

> Apresentação interativa de consultoria em segurança de redes, comparando três abordagens de proteção de tráfego: **Conexão Direta**, **Proxy** e **VPN**.

##  Sobre o Projeto

Esta consultoria foi criada pela **Master Security** com o objetivo de orientar empresas na escolha da melhor solução de segurança para sua infraestrutura de rede.

A apresentação cobre:
- Como cada tecnologia funciona em linguagem simples
- O que é exposto ou protegido em cada cenário
- Comparativo técnico: proteção, criptografia e velocidade
- Por que a **VPN é a solução recomendada**
